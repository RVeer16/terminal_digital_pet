# Python Digipet

A terminal-based virtual pet built in Python that reacts to your productivity.

## How it works
- Your pet's health changes based on how productive you are
- Complete tasks in time and your pet gets healthier
- Neglect your tasks and your pet gets sad or dies

## Features
- Talk to your pet and it responds in its own language
- Built in diary that saves every conversation with date
- Task management system with time tracking
- Pet mood changes based on health (happy, fine, sad, dead)
- Read previous diary entries anytime

## Pet health
- Above 75 — Happy
- 50 to 75 — Fine
- Below 50 — Sad
- 0 or below — Dead

## Requirements
No external libraries needed. Uses Python built-in modules only:
json, random, datetime, os, time
